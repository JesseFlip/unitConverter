Universal Unit Converter
Problem: Needed a fast, mobile-friendly tool for converting units across multiple categories (length, volume, mass, temperature, speed, area, time, digital storage).
Solution: Built a responsive web application with real-time conversion, dark mode support, and SEO optimization.
Impact: Live production app serving users worldwide. Demonstrates front-end development, UX design, and deployment automation.
Tech Stack: HTML, CSS, JavaScript, Netlify
